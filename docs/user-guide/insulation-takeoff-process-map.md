# Insulation Takeoff Process Map

Use this as the quick training handout. The long operator guide explains the details; this process map shows the order of work and the checkpoints that prevent bad estimates.

## Process

1. Setup: confirm company profile, logo, quote terms, and tax settings.
2. Project: create/select client, create project, upload plan PDF, open Takeoff.
3. Vision: AI reads the plan pages, suggests page roles, and extracts useful clues; user confirms Primary Takeoff and Support Page selections.
4. Areas: AI suggests likely takeoff areas and best pages; user calibrates, traces each area, chooses the zone type, and fills area details.
5. Calibration: user clicks Scale or Cal., clicks both endpoints of a known dimension, enters the real printed length, and verifies the scale.
6. Measure: user traces walls/surfaces/roof areas; AI can scan selected window notes, door notes, and roof pitch when requested.
7. Review: the app builds worksheet rows from measured scope and scanned clues; user checks quantities, units, specs, deductions, and manual rows.
8. Quote: confirm pricing, tax, terms, included rows, generate the quote, and download the PDF.

## Calibration Mini Map

1. Pick a long printed dimension.
2. Click Scale or Cal.
3. Click endpoint A.
4. Click endpoint B.
5. Enter the real length.
6. Confirm Scale on.
7. Compare one known dimension before continuing.

## Measuring Mini Map

1. Select the area.
2. Pick the correct tool.
3. Trace the scope.
4. Finish the trace with Enter or double-click.
5. Add windows and doors to the correct wall.
6. Review quantities before quoting.

## Stop And Fix When

- The page is not calibrated.
- A known length does not match after calibration.
- A wall is attached to the wrong area.
- A window or door is not tied to the correct wall.
- A worksheet row has the wrong unit, R-value, or quantity.

## AI Rule

AI suggests page roles, support evidence, area hints, opening sizes, and roof pitch. The user still confirms every page, calibration, traced area, opening, worksheet row, and quote.
